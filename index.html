Bandung Lautan Api

Bandung Jawa Barat, 24 Maret 1946

Penjajah yang hendak menguasai Bandung dalam peristiwa Bandung Lautan Api adalah tentara Belanda (NICA) dan Sekutu. Pasukan Sekutu/Inggris tiba di Bandung pada 12 Oktober 1945.

Apasih tujuan utamanya?

Tujuan utama Bandung Lautan Api adalah untuk menghalangi
Sekutu menggunakan Bandung sebagai markas militer strategis.
Peristiwa ini juga merupakan bentuk perlawanan rakyat terhadap
penjajahan.

Pro Characters And Detractor

Strategi perang yang digunakan dalam Bandung Lautan Api adalah taktik bumi hangus. Taktik ini dilakukan oleh TRI dan laskar-laskar pemuda Priangan.

Bagaimana taktik bumi hangus dilakukan? Membakar wilayah dan melakukan pengeboman di titik-titik penting di Bandung Membakar markas dan asrama TRI. Membakar rumah-rumah warga

Hasil Pertempuran

Pertempuran Bandung Lautan Api pada 24 Maret 1946 berdampak besar bagi Indonesia dan Sekutu. Rakyat dan tentara Indonesia membakar Bandung untuk mencegah penguasaan Sekutu, mengakibatkan penipisan persediaan senjata dan banyaknya tentara Belanda yang tewas. Sekutu gagal menguasai kota, sementara masyarakat bersatu dalam perjuangan. Peristiwa ini diabadikan dalam seni dan dianggap sebagai strategi bumi hangus yang efektif, dengan sekitar 200.000 hingga 500.000 warga dievakuasi.

Moral Values

Peristiwa Bandung Lautan Api yang terjadi pada 23 Maret 1946 mengajarkan nilai-nilai kepahlawanan, seperti Rela berkorban, Semangat persatuan, Kesadaran nasionalisme, Keberanian, Semangat juang
